# ⚙️ Setup Guide

This guide explains how to configure and run the **NEXT-KIETO-UI** project locally.

---

## 📦 Prerequisites

- Node.js v20+  
- npm v9+  
- Git installed  

---

## 🚀 Installation

### 1. Clone the repository
```bash
git clone https://github.com/Adyllsxn/next-kieto-ui.git
```

### 2. Enter the project folder
```bash
cd next-kieto-ui/web
```

### 3. Install dependencies
```bash 
npm install
```

### 4. Run the project

#### 4.1. Development environment
```bash 
npm run dev
```

#### 4.2. Production build
```bash 
npm run build
npm run start
```

### 5. Tests and Lint

#### 5.1. Run tests
```bash 
npm test
```

#### 5.2. Run lint
```bash 
npm run lint
```
---
